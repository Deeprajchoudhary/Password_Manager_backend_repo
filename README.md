# <PassOP/> - API Backend

The backend server for the PassOP Password Manager. This application handles secure connections to MongoDB Atlas and provides RESTful API endpoints for the frontend interface.

## ✨ Features
* **Seamless Database Integration:** Connects directly to MongoDB clusters using the official Node driver.
* **CORS Enabled:** Safely accepts cross-origin requests from your frontend client.
* **Environment Security:** Protects database credentials using `dotenv`.

## 🛠️ Tech Stack
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB
* **Middleware:** Cors, Body-Parser, Dotenv
